# ⚡ HƯỚNG DẪN NHANH - 4 BƯỚC

## 🎯 Mục tiêu: Game online 24/7, tắt máy vẫn chơi được

---

## BƯỚC 1: TẠO GITHUB REPO (2 phút)

1. Vào: **https://github.com/new**
2. Repository name: `one-piece-arena`
3. Chọn **Public**
4. Click **Create repository**
5. Copy URL: `https://github.com/YOUR_USERNAME/one-piece-arena.git`

---

## BƯỚC 2: PUSH CODE (1 phút)

Mở **Command Prompt**, chạy từng lệnh:

```bash
cd "C:\Users\MINH HIEU\Downloads\lap trinh abk\one-piece-arena"

git remote add origin https://github.com/YOUR_USERNAME/one-piece-arena.git

git branch -M main

git push -u origin main
```

**Thay YOUR_USERNAME bằng username GitHub của bạn!**

---

## BƯỚC 3: DEPLOY LÊN RAILWAY (2 phút)

1. Vào: **https://railway.app**
2. Click **Login** → **Login with GitHub**
3. Click **New Project**
4. Chọn **Deploy from GitHub repo**
5. Chọn repo **one-piece-arena**
6. Đợi deploy xong (~1-2 phút)

---

## BƯỚC 4: TẠO URL (30 giây)

1. Click vào project trong Railway
2. Click tab **Settings**
3. Scroll xuống **Networking**
4. Click **Generate Domain**
5. Copy URL: `https://one-piece-arena-production.up.railway.app`

---

## ✅ XONG!

**URL game của bạn:**
```
https://one-piece-arena-production.up.railway.app
```

**Share URL này để mọi người chơi!**
